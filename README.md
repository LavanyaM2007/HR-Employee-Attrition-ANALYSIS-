# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition data to identify the factors responsible for employee turnover.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Cleaning
- Attrition Rate Analysis
- Department-wise Analysis
- Salary Analysis
- Experience Analysis
- Data Visualization

## Key Findings
- Overtime employees show higher attrition.
- Low salary employees leave more frequently.
- Job satisfaction affects attrition.
- Experience influences employee retention.
